Nota.py 
================

    Um programa que calcula a media de uma escola 


----

Executar
----------

Efetuar a execução do código-fonte:


        $ python3 nota.py


----

Uso 
---

    Calcula a media de uma escola

----

Exemplos
--------

       1 bimestre:2
       2 bimestre:2
       3 bimestre:2
       4 bimestre:2
    
       vocâ reprovou
